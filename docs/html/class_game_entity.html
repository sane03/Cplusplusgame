Detailed Description

The purpose of this class is to load game entity images with the use of the ImageHandler object. This class also provides access to the sprite of the game entity, game constants, game entity states and scaling characteristics. Inheritance was extensively implemented in the design architecture and there this class serves as the main base class for all game entities.
